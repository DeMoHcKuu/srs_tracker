# SRS Study Tracker

Веб-сервис интервального повторения для эффективного запоминания информации. Система автоматически рассчитывает даты следующего повторения карточек на основе алгоритма SM-2 и помогает отслеживать прогресс через интерактивную аналитику.

**Ссылка на рабочий проект:** https://demohckuu.pythonanywhere.com

## Технологии
- **Backend**: Python 3.13, Django 5.2
- **Database**: SQLite (разработка), PostgreSQL (production)
- **Analytics & Visualization**: Plotly (интерактивные графики)
- **Frontend**: Bootstrap 5.3, HTML/CSS
- **Deployment**: PythonAnywhere

## Как запустить проект локально

### Требования
- Python 3.10+
- pip (менеджер пакетов)

### Пошаговая инструкция

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/demohckuu/srs-tracker.git
   cd srs-tracker
2. **Создайте и активируйте виртуальное окружение:**

bash
python -m venv venv
source venv/bin/activate       # Linux/Mac
venv\Scripts\activate           # Windows

3. **Установите зависимости:**

bash
pip install -r requirements.txt

4. **Выполните миграции:**

bash
python manage.py migrate

5. **Создайте суперпользователя (администратора):**

bash
python manage.py createsuperuser

6. **(Опционально) Загрузите тестовые данные:**

bash
python manage.py load_sample_data

7. **Запустите сервер:**

bash
python manage.py runserver

8. **Откройте проект в браузере:**

Главная: http://127.0.0.1:8000/
Вход: http://127.0.0.1:8000/accounts/login/
Админка: http://127.0.0.1:8000/admin/

### Создание колоды
Авторизуйтесь на сайте.

Перейдите в Колоды.

Нажмите + Создать.

Введите название и описание, сохраните.

### Добавление карточек
Откройте колоду.

Нажмите + Добавить карточку.

Введите вопрос (front_text) и ответ (back_text).

Опционально выберите теги.

Сохраните.

### Повторение карточек
Перейдите на Повторить.

Смотрите вопрос, нажимайте "Показать ответ".

Оцените свой ответ (0–5), где:

0–2: не помню (интервал = 1 день).

3–5: помню (интервал увеличивается по SM-2).

Система автоматически рассчитает дату следующего повторения.

### Просмотр аналитики
Перейдите на Аналитика.

Посмотрите графики прогресса за последние 30 дней.

Найдите "сложные" карточки (низкая средняя оценка).

### Алгоритм SM-2
Система использует алгоритм SuperMemo-2 для расчёта интервалов повторения:

После первого повторения: интервал = 1 день.

После второго: интервал = 6 дней.

Далее: интервал = (предыдущий интервал) × (коэффициент лёгкости).

Коэффициент лёгкости корректируется на основе качества ответа.
